# 🤖 GymProject - Servizio AI

## ⚡ Avvio del Servizio AI

**UNA sola cosa da fare:**

```
Doppio click: AI/Setup-GymProject.bat
```

Questo installa tutto, crea l'icona desktop e avvia il servizio AI.

---

## 🖥️ Frontend React (Interfaccia Web)

```powershell
cd palestra-frontend
npm run dev
```

Apri: http://localhost:5173
